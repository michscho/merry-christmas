# 🎄 Frohe Weihnachten 2025

Eine kleine interaktive Weihnachts-Webseite. Reines HTML/CSS/JavaScript,
ohne Build-Schritt.

## Lokal starten
`index.html` im Browser öffnen – keine Installation nötig.

## Projektstruktur
```
index.html       → HTML-Markup
css/styles.css   → Styles
js/app.js        → JavaScript-Logik
assets/          → Bilder & Medien
```
